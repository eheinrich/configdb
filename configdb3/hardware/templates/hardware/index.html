API

Configdb3 provides a RESTful api to query data. The data will be returned in JSON format unless accessed via browser, in which case a user friendly interface will be provided.

The top level resource are:

  • Sites: {% url 'site-list' %}
  • Enclosures: {% url 'enclosure-list' %}
  • Telescopes: {% url 'telescope-list' %}
  • Instruments: {% url 'instrument-list' %}
  • Cameras: {% url 'camera-list' %}
  • Modes: {% url 'mode-list' %}
  • Filter Wheels: {% url 'filterwheel-list' %}

Individual resources can be accessed by id:

  • Maui: {% url 'site-detail' pk=7 %}
  • kb34: {% url 'camera-detail' pk=134 %}

Resources can also be filtered by properties:

  • Site(s) with code ogg: {% url 'site-list' %}?code=ogg
  • 1 meter telescopes: {% url 'telescope-list' %}?code=1m0a
